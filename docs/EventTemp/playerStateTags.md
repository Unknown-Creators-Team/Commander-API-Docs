---
title: "プレイヤー状態タグ（tags.ts）"
---

## 説明
プレイヤーの様々な状態を毎ティック監視し、対応するタグを自動的に付与/削除するイベント群です。

---

## プレイヤー権限

### isOp（オペレーター）
| タグ名 | 説明 |
| ----- | ----- |
| `capi:op` | プレイヤーがオペレーター権限を持っている場合に付与されます。 |

## プレイヤーの移動状態

## 使用例

飛行中のプレイヤーを検出する例：
```mcfunction
execute as @a[tag=capi:fly] run say 飛行中です！
```

モバイルプレイヤーを検出する例：
```mcfunction
execute as @a[tag=capi:mobile] run say モバイルでプレイしています！
```

スニーク中のプレイヤーを検出する例：
```mcfunction
execute as @a[tag=capi:sneak] run say スニーク中です！
```
