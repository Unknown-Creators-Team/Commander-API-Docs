---
title: "rotation（回転）"
---

## 説明
プレイヤーの視点の回転角度を毎ティック取得するイベントです。

---

## トリガースコア
毎ティック自動で更新されるスコアボードです。

| スコアボード名 | 説明 |
| ----- | ----- |
| `capi:rotation_x` | プレイヤーの上下の回転角度（ピッチ）です。 |
| `capi:rotation_y` | プレイヤーの左右の回転角度（ヨー）です。 |

## 使用例
特定の方向を向いているプレイヤーを検出する例：

```mcfunction
execute as @a[scores={capi:rotation_y=0..90}] run say 東を向いています！
```
