---
title: "velocity（速度）"
---

## 説明
プレイヤーの速度を毎ティック取得するイベントです。X、Y、Z軸それぞれの速度と、合成速度を取得できます。

---

## トリガースコア
毎ティック自動で更新されるスコアボードです。

| スコアボード名 | 説明 |
| ----- | ----- |
| `capi:velocity_x` | プレイヤーのX軸方向の速度です。（×200） |
| `capi:velocity_y` | プレイヤーのY軸方向の速度です。（×200） |
| `capi:velocity_z` | プレイヤーのZ軸方向の速度です。（×200） |
| `capi:velocity_xz` | プレイヤーの水平方向（XZ平面）の速度です。（×200） |
| `capi:velocity_xyz` | プレイヤーの3次元空間での合成速度です。（×200） |

## 注意事項
- 速度の値は200倍されて保存されます。
- 実際の速度を取得するには、スコアの値を200で割る必要があります。

## 使用例
高速で移動しているプレイヤーを検出する例：

```mcfunction
execute as @a[scores={capi:velocity_xz=100..}] run say 速く移動しています！
```
