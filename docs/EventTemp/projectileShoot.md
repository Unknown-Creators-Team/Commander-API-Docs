---
title: "projectileShoot"
---

## 説明
プレイヤーが飛び道具を発射したときにトリガーされるイベントです。弓やクロスボウで矢を発射した際に検出されます。

---

## トリガータグ
イベントがトリガーされたとき、プレイヤーに自動付与されるタグです。

| タグ名 | 説明 |
| ----- | ----- |
| `capi:shot` | このイベントがトリガーされたとき、プレイヤーに自動付与されます。 |
| `shot.with:{飛び道具ID}` | 発射した飛び道具のIDがタグとして付与されます。 |
| `shot.from:{プレイヤー名}` | 発射したプレイヤーの名前がタグとして付与されます。 |

## 対応アイテム
- 弓（minecraft:bow）
- クロスボウ（minecraft:crossbow）

## 使用例
矢を発射したプレイヤーを検出する例：

```mcfunction
/execute as @a[tag=capi:shot] run say 矢を発射しました！
```
