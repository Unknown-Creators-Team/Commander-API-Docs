## 概要

**test** は、テストを実行する ScriptEvent です。  
Commander-APIが正しく動作していることを確認するためのコマンドです。

## 構文

```mcfunction
/scriptevent capi:test <テスト名>
```

## パラメータ

| パラメータ | 説明 | 必須 |
|---|---|---|
| `テスト名` | 実行するテストの名前（省略可）  | ✓ |

## 使用例

### 全てのテストを実行

テスト名を省略すると、**全てのテスト**が実行されます。
:::danger 建築物が破壊される可能性があります。
先程も述べましたが、**全てのテスト**を実行すると多くの範囲を利用するため、建築物が破壊される可能性があります。  
**必ず**周囲に大きな土地を確保した上で実行してください。

```mcfunction
/scriptevent capi:test
```
:::

### 特定のテストを実行

特定のテストのみを実行します。

```mcfunction
/scriptevent capi:test テスト名
```

| テスト名 | 説明 | 種類 |
|---|---|---|
|`button_push`|プレイヤーがボタンを押したときのイベントをテストします|イベント|


## 注意事項

- 実行者または実行位置が必要です。
- 実行環境には**非常に注意して**行うようにしてください。

## 関連項目

このコマンドは主に開発者向けの機能です。
