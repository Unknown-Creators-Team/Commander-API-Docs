実行者のスコアを代入します。

テキストが使用できる場所で以下のように使用します。
```
{score:スコア名}
```
`スコア名` にはスコアの名前を入れます。
`スコア名` が `Capi:health` で、実行者の体力がハート10個分の場合、以下のように表示されます。
```
20
```

### 例

+++ tell
tellメソッドを使用した例です。
自分の体力を表示します。
```
/tag @s add "tell:{score:Capi:health} HP"
```
出力
```
20 HP
```
+++ rename
renameメソッドを使用した例です。
自分の体力を表示します。
```
/tag @s add "rename:{name}{nl}{score:Capi:health} HP"
```
Notchを実行者とし、体力はハート10個の場合、以下のように表示されます。
```
Notch
20 HP
```
[!ref name変数](./name.md)
[!ref nl変数](./nl.md)
+++

## 実行者以外のスコアを代入
score変数では、実行者以外のスコアを代入することもできます。

!!!warning 注意
オフラインのプレイヤーのスコアを取得することはできません。
!!!

テキストが使用できる場所で以下のように使用します。
```
{score:{name=名前,object=スコア名}}
```
`名前` にはプレイヤーの名前を入れます。
`スコア名` にはスコアの名前を入れます。<br />
`名前` が `Notch` で、`スコア名` が `Capi:health` で、Notchの体力がハート10個分の場合、以下のように表示されます。
```
20
```
