プレイヤーがターゲットブロックを発動させたときにトリガーします。
## tags
* **Capi:target**
> このイベントがトリガーされたときプレイヤーに追加されます。

## scores
* **Capi:targetX**
> ターゲットブロックのX座標が代入されます。

* **Capi:targetY**
> ターゲットブロックのY座標が代入されます。

* **Capi:targetZ**
> ターゲットブロックのZ座標が代入されます。

* **Capi:targetPower**
> ターゲットブロックのパワーが代入されます。
