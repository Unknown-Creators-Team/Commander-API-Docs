プレイヤーがワールドに参加したときにトリガーされます。
## tags
* **Capi:join**
> このイベントがトリガーされたときプレイヤーに追加されます。
## scores
* **Capi:joinCount**
> プレイヤーがワールドに参加したときに1加算されます。
* **Capi:playerJoinX**
> プレイヤーがワールドに参加したときのX座標が代入されます。
* **Capi:playerJoinY**
> プレイヤーがワールドに参加したときのY座標が代入されます。
* **Capi:playerJoinZ**
> プレイヤーがワールドに参加したときのZ座標が代入されます。
