# Commander API ツール

Commander API の開発を支援するための便利なツール集です。

## 利用可能なツール

### [Command Splitter](./calls)
長いテキストを [Call](../docs/ScriptEvent/call) コマンドで使用できる形式に分割するツールです。テキストを適切な長さに分割し、複数のコマンドに変換します。

### [ESON - JSON Converter](./eson-json)
ESON（拡張 JSON）形式と標準 JSON 形式を相互に変換するツールです。Commander API で使用する設定ファイルの作成や編集に便利です。

### [Form Creator](./form-editor)
Commander API のフォーム（アクション、メッセージ、モーダル）を視覚的に作成・編集できるツールです。ドラッグ＆ドロップで要素を配置し、ESON/JSON コマンドを自動生成します。

## ツールの使い方

各ツールページでは、インタラクティブなエディタを使用して直接作業できます。生成されたコードは、コピーボタンでクリップボードにコピーして、すぐにプロジェクトで使用できます。
